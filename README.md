# SkiOffBounds
For running the Django project
```
cd SkiOffBounds
py manage.py runserver
```
For user view: [http://127.0.0.1:8000/](http://127.0.0.1:8000/home)
For admin functionalities: [http://127.0.0.1:8000/admin](http://127.0.0.1:8000/admin)


To add initialization data if the database is empty
```
python manage.py loaddata datos_iniciales
```
